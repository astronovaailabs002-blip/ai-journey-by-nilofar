# AI Journey by Nilofar

Premium multi-page personal portfolio for Nilofar Naaz.

## Pages

- `index.html` - Home
- `about.html` - About
- `skills.html` - Skills
- `projects.html` - Projects
- `journey.html` - AI Journey timeline
- `contact.html` - Contact

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

No frameworks or libraries are used.

## Features

- Mobile-first responsive layout
- Luxury purple glassmorphism UI
- Animated aurora CSS background
- Sticky responsive navbar with hamburger menu
- Scroll progress bar
- Custom loader
- Cursor glow on pointer devices
- Scroll reveal animations
- Back-to-top button
- SEO and social sharing metadata
- Accessible focus states and semantic structure
